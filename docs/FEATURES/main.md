# 메인 화면 (Main Screen)

## 개요

게임 실행 시 가장 먼저 보여지는 첫 화면으로, 플레이어가 게임을 시작하거나 기본 정보를 확인할 수 있는 진입점이다.

## 화면 구성 (초안)

- **타이틀 로고**: 화면 상단 중앙에 "PANG" 게임 타이틀 표시
- **시작 버튼**: "게임 시작" (스페이스바 또는 클릭/터치로 진행)
- **조작 안내**: 이동(←/→), 발사(스페이스 또는 클릭) 등 기본 조작키 안내 텍스트
- **최고 기록(옵션)**: 최고 점수 또는 최근 클리어한 미션 표시 (추후 확장)
- **배경**: 풍선이 통통 튀는 애니메이션 등 간단한 배경 연출 (추후 확장)

## 화면 흐름

1. 게임 로드 완료 → 메인 화면 표시
2. 플레이어가 시작 입력 → Mission 1로 진입
3. (추후) 설정/스테이지 선택 등 서브 메뉴 진입 가능

## 범위

- 1차 구현은 타이틀 텍스트 + 시작 버튼 + 조작 안내 정도의 최소 구성으로 한정
- 사운드, 애니메이션, 스테이지 선택 UI 등은 이후 반복(iteration)에서 추가
