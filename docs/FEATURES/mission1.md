# Mission 1 - 난이도 및 규칙

`docs/PRD.md`의 Mission 1 MVP 범위를 기반으로 실제 구현 시 적용할 난이도와 세부 규칙을 정의한다.

## 목표

- Mission 1은 팡 게임의 기본 규칙을 검증하는 튜토리얼 성격의 첫 스테이지
- 화면 내 모든 풍선을 제거하면 클리어

## 난이도 구성 (초안)

- 등장 풍선 수: 소수 (예: 큰 풍선 1~2개로 시작)
- 풍선 이동 속도: 느림 (초기 학습 난이도를 낮춤)
- 풍선 크기 단계: 4단계 (큰 → 중간 → 작은 → 소멸), `game_rule.md`의 분열 규칙을 그대로 적용
- 파워업: 미등장 (Mission 1에서는 파워업 없이 기본 조작/규칙만으로 클리어하도록 구성)

## 플레이어 초기 상태

- HP: 10
- 목숨(Life): 3개
- 기본 작살 1발 발사

## 클리어 조건

- 배치된 모든 풍선(분열된 풍선 포함)을 전부 제거

## 실패 조건

- 목숨을 모두 소진하면 게임 오버, Mission 1부터 재시작

## 구현 범위 (MVP)

- 좌우 이동 및 작살 발사 조작
- 풍선 분열(4단계) 및 소멸 로직
- 플레이어-풍선 충돌 판정 및 HP/목숨 감소
- 모든 풍선 제거 시 클리어 판정 및 결과 화면 전환
- 점수 시스템, 보너스 라이프, 파워업 등은 Mission 2 이후로 확장 예정
